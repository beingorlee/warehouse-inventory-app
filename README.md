# 仓库库存管理系统

一个基于Android原生开发的仓库库存管理应用，支持多层仓库的可视化管理和产品增删改查操作。

## 功能特性

### 🏢 仓库管理
- 支持多层仓库设置（默认1层，可自定义添加）
- 每层采用标准矩形布局：左区(40%) + 过道(10%) + 右区(40%)
- 灵活的列数配置，左右区域可独立设置列数
- 每列支持多个栈板位置，每个栈板可存放多种产品

### 📦 产品管理
- 产品型号验证：字母开头，5-7位，支持字母、数字、连字符
- 自动格式化：去除空格，转换为大写
- 数量验证：必须为正整数
- 位置标识：如"L5-2"表示左区第5列第2个栈板

### 🔍 查询功能
- 按产品型号快速查询
- 显示产品所在楼层、位置和数量
- 可视化平面图展示，直观显示产品位置
- 支持点击栈板查看详细信息

### 📱 用户界面
- Material Design 3设计风格
- 响应式布局，适配不同屏幕尺寸
- 直观的平面图显示，支持缩放和滚动
- 简洁清晰的操作界面

### 🔒 数据安全
- 本地SQLite数据库存储
- 重置操作需二次确认（输入"确认"）
- 数据持久化，应用重启后数据不丢失

## 技术架构

### 开发技术栈
- **语言**: Kotlin
- **架构**: MVVM (Model-View-ViewModel)
- **数据库**: Room (SQLite)
- **UI框架**: Android View System + Material Design 3
- **导航**: Navigation Component
- **异步处理**: Kotlin Coroutines + LiveData

## 使用流程

### 1. 初始化设置
1. 首次打开应用，进入仓库设置页面
2. 设置仓库层数（1-10层）
3. 为每层配置左右区域的列数（1-20列）
4. 完成设置后进入主界面

### 2. 产品管理
1. **添加产品**: 点击"添加产品"按钮，填写产品信息
2. **查询产品**: 输入产品型号进行搜索
3. **查看详情**: 点击平面图中的栈板查看产品详情
4. **更新库存**: 通过产品详情页面修改数量

### 3. 平面图操作
1. 选择楼层标签切换不同楼层
2. 平面图显示实际的仓库布局
3. 栈板颜色和标签显示产品信息
4. 支持滚动查看大型仓库布局

## 数据模型

### 产品 (Product)
- 型号: 5-7位字符，字母开头
- 数量: 正整数
- 楼层: 所在楼层编号
- 位置: 栈板位置标识

### 楼层 (Floor)
- 楼层编号: 唯一标识
- 左区列数: 1-20列
- 右区列数: 1-20列

### 栈板 (Pallet)
- 位置标识: L/R + 列号 + "-" + 行号
- 所属楼层: 楼层编号引用

## 安装要求

- Android 7.0 (API 24) 或更高版本
- 最小内存: 2GB RAM
- 存储空间: 50MB

## 开发环境

- Android Studio Hedgehog | 2023.1.1 或更高版本
- Kotlin 1.9.10
- Gradle 8.2.0
- compileSdk 34
- targetSdk 34

## 构建说明

1. 克隆项目到本地
2. 使用Android Studio打开项目
3. 等待Gradle同步完成
4. 连接Android设备或启动模拟器
5. 点击运行按钮构建并安装应用
